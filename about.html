About MIDI Talk in a Box

This is a music hack created at Music Hack Day Boston 2014 by Paul Lamere. It is based on Girl Talk in a box. It is powered by The Echo Nest. This hack was created using the same technology as the Infinite Jukebox, Infinite Gangnam Style and Bohemian Rhapsichord.

The app will load up a song and cut it into beat aligned blocks of audio and represent each of these blocks with a colorful visualization. You can then interact with the song as it plays using a whole bunch of launchpad and keyboard controls. The beats are colored based upon their timbre, so similar sounding beats will have similar colors.
